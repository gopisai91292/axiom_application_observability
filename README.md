# Axiom Application – Observability & SLO Design

---

## Overview
This repository contains the deliverables for the Axiom Observability & SLO Design assignment.  
It outlines an observability architecture for a distributed financial platform running across AWS and on-prem environments.

The design focuses on:
- Early detection of service degradation  
- End-to-end tracing across services  
- SLO-driven alerting and reliability governance  
- Cost-efficient scalability and retention
